# Eleagues App

This Application is developed to manage leaderboards and stats of Cricket Leagues that are played on Playstation.

It's hosted as a Live Demo here: [Eleagues](https://ngcore.eleagues.net/)

### Backend:
- .Net Core 3.0
- EntityFrameworkCore
- Sqlite
- Jwt
- CloudinaryDotNet
- AutoMapper

### Frontend:
- Angular 8.2
- Typescript 3.5.3
- angular-jwt
- Bootstrap 4
- NGX-Bootstrap
- Font-Awesome
- Rxjs
- Alertifyjs
- flag-icons
- time-ago-pipe
